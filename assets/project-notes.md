# Notatki projektowe

## Cel

Stworzyć MVP strony dla developera, inspirowanej filmem, gdzie dom jest pokazany jako interaktywna scena 3D, a scroll prowadzi użytkownika przez kolejne etapy prezentacji.

## Założenia MVP

- bez tekstur,
- biała makieta architektoniczna,
- proste bryły zamiast docelowego modelu,
- kamera sterowana scrollem,
- przejście: zewnętrzny widok inwestycji -> wejście -> salon -> kuchnia -> hall -> strefa prywatna,
- responsywny overlay z opisami etapów.

## Docelowy efekt produkcyjny

Po akceptacji kierunku MVP można przygotować wersję z:

- realistycznym modelem budynku,
- animowanymi światłami,
- materiałami elewacji i wnętrz,
- trybem wyboru pomieszczeń,
- widokiem rzutów 2D,
- integracją z CMS.
